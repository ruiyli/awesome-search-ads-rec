# Awesome search, ads, and recommendation systems weekly

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

A curated weekly digest of the latest research papers across **Search**, **Advertising**, and **Recommendation** systems. This repository aims to help researchers, engineers, and practitioners stay up-to-date with cutting-edge developments in these three closely related fields.

We cover a broad range of topics including but not limited to:

- **Search**: query understanding, learning to rank, dense retrieval, question answering, and search evaluation
- **Advertising**: computational advertising, bid optimization, click-through rate prediction, auction mechanisms, and ad targeting
- **Recommendation**: collaborative filtering, sequential recommendation, conversational recommendation, graph-based methods, and large language models for recommendation

Whether you are an academic researcher exploring new directions or an industry practitioner seeking practical insights, this weekly collection is designed to be your go-to resource.

## Latest Weekly Update

> Week of 2026-02-23

- [ ] **Title**: [Reasoning-Enhanced Dense Retrieval via Contrastive Chain-of-Thought Distillation](https://arxiv.org/abs/2602.00001)  
  **Authors**: Yiming Zhang, Wei Chen, Jiafeng Guo, Xueqi Cheng  
  **Venue**: WWW 2026  
  **Summary**: Proposes a contrastive chain-of-thought distillation framework that transfers reasoning capabilities from large language models to dense retrieval models, improving passage ranking on MS MARCO and BEIR benchmarks.

- [ ] **Title**: [Adaptive Query Rewriting with Multi-Granularity Feedback for Conversational Search](https://arxiv.org/abs/2602.00002)  
  **Authors**: Hao Li, Zhiyuan Liu, Maosong Sun  
  **Venue**: SIGIR 2026  
  **Summary**: Introduces a multi-granularity feedback mechanism for query rewriting in conversational search, leveraging token-level and passage-level signals to reduce topic drift across dialogue turns.
