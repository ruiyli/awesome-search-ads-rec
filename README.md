# Awesome search ads recommendation systems weekly

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

A curated weekly digest of the latest research papers across **Search**, **Advertising**, and **Recommendation** systems. This repository aims to help researchers, engineers, and practitioners stay up-to-date with cutting-edge developments in these three closely related fields.

We cover a broad range of topics including but not limited to:

- **Search**: query understanding, learning to rank, dense retrieval, question answering, and search evaluation
- **Advertising**: computational advertising, bid optimization, click-through rate prediction, auction mechanisms, and ad targeting
- **Recommendation**: collaborative filtering, sequential recommendation, conversational recommendation, graph-based methods, and large language models for recommendation

Whether you are an academic researcher exploring new directions or an industry practitioner seeking practical insights, this weekly collection is designed to be your go-to resource.

## Latest Weekly Update

> Week of 2026-02-23

### Search

- [ ] **Title**: [Reasoning-Enhanced Dense Retrieval via Contrastive Chain-of-Thought Distillation](https://arxiv.org/abs/2602.00001)  
  **Authors**: Yiming Zhang, Wei Chen, Jiafeng Guo, Xueqi Cheng  
  **Venue**: WWW 2026  
  **Summary**: Proposes a contrastive chain-of-thought distillation framework that transfers reasoning capabilities from large language models to dense retrieval models, improving passage ranking on MS MARCO and BEIR benchmarks.

- [ ] **Title**: [Adaptive Query Rewriting with Multi-Granularity Feedback for Conversational Search](https://arxiv.org/abs/2602.00002)  
  **Authors**: Hao Li, Zhiyuan Liu, Maosong Sun  
  **Venue**: SIGIR 2026  
  **Summary**: Introduces a multi-granularity feedback mechanism for query rewriting in conversational search, leveraging token-level and passage-level signals to reduce topic drift across dialogue turns.

- [ ] **Title**: [Unified Sparse and Dense Retrieval with Learned Index Routing](https://arxiv.org/abs/2602.00003)  
  **Authors**: Xueguang Ma, Jimmy Lin  
  **Venue**: ECIR 2026  
  **Summary**: Presents a learned routing strategy that dynamically selects between sparse (BM25) and dense retrieval indexes per query, achieving improved efficiency-effectiveness trade-offs on large-scale collections.

### Advertising

- [ ] **Title**: [Budget-Constrained Autobidding with Reinforcement Learning in Real-Time Auctions](https://arxiv.org/abs/2602.00004)  
  **Authors**: Yunfan Zhao, Jun Wang, Weinan Zhang  
  **Venue**: KDD 2026  
  **Summary**: Formulates autobidding as a constrained Markov decision process and proposes a Lagrangian-based reinforcement learning algorithm that optimizes advertiser ROI under strict budget constraints in real-time bidding.

- [ ] **Title**: [Causal Debiasing for Click-Through Rate Prediction in Sponsored Search](https://arxiv.org/abs/2602.00005)  
  **Authors**: Peng Wu, Kuang-chih Lee, Xiao Yang  
  **Venue**: WSDM 2026  
  **Summary**: Applies causal inference techniques to mitigate position bias and selection bias simultaneously in click-through rate prediction models for sponsored search ads, demonstrating gains on both offline metrics and online A/B tests.

- [ ] **Title**: [Cross-Platform Ad Creative Generation via Vision-Language Pre-Training](https://arxiv.org/abs/2602.00006)  
  **Authors**: Mingyang Li, Yong Liu, Rui Zhang  
  **Venue**: AAAI 2026  
  **Summary**: Proposes a vision-language pre-training framework for automatically generating ad creatives that adapt to different platform formats, reducing manual design effort while maintaining click-through performance.

### Recommendation

- [ ] **Title**: [LLM-Augmented Sequential Recommendation with Interest-Aware Prompt Tuning](https://arxiv.org/abs/2602.00007)  
  **Authors**: Jianxin Chang, Peng Zhou, Xiangnan He  
  **Venue**: WWW 2026  
  **Summary**: Integrates large language models into sequential recommendation through interest-aware soft prompts, capturing long-term user preferences more effectively than conventional ID-based sequential models.

- [ ] **Title**: [Graph Contrastive Learning for Cold-Start Recommendation with Auxiliary Knowledge](https://arxiv.org/abs/2602.00008)  
  **Authors**: Yuxiao Dong, Jie Tang, Nitesh Chawla  
  **Venue**: ICDE 2026  
  **Summary**: Addresses the cold-start problem by augmenting user-item interaction graphs with external knowledge graphs and applying graph contrastive learning to learn robust representations for new users and items.

- [ ] **Title**: [Multi-Objective Optimization for Fairness-Aware Recommendation](https://arxiv.org/abs/2602.00009)  
  **Authors**: Yao Wu, Jian Tang, Lina Yao  
  **Venue**: RecSys 2026  
  **Summary**: Introduces a Pareto-optimal multi-objective framework that jointly optimizes recommendation accuracy and multiple fairness criteria across user groups, providing controllable trade-off solutions for platform operators.
