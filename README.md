A curated weekly digest of the latest research papers across **Search**, **Advertising**, and **Recommendation** systems.

This repository helps researchers, engineers, and practitioners stay up-to-date with cutting-edge developments in these closely related fields. We cover topics including query understanding, learning to rank, dense retrieval, click-through rate prediction, bid optimization, collaborative filtering, sequential recommendation, large language models for recommendation, and more.
